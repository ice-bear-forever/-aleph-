help

main menu

The main menu is the first panel you see when aleph is opened. From here you can start new quizzes, load backups, and continue or cancel any in-progress quizzes. It can be accessed at any time by clicking the aleph logo in the top left corner of the window.

starting quizzes

To start a quiz, choose a question set from the dropdown menu. Once a question set is selected, press start to begin a quiz.

backups

If a backup of an in-progress quiz is available, it can be loaded by pressing the backup option below the start button.

in-progress quizzes

When a quiz is in progress, you can continue or cancel it from the main menu.

quizzes

Quizzes are started from the main menu. Every quiz consists of 5 randomly selected questions from the chosen set. Options for multiple-choice and multiple-answer questions are also listed in a random order.

question types

aleph supports four types of questions—multiple-choice, true-or-false, short-answer, and multiple-answer.

auto-saving and backups

aleph automatically saves quiz progress when you switch questions or exit the quiz panel. If aleph is exited for any reason, including by using the stop quiz button, the last saved backup can be loaded from the main menu. Elapsed time, current question, and previous answers will be restored. Backups are deleted when their quizzes are completed.

results

When you finish a quiz, aleph displays its results. To print, click the button in the top right corner of the window.

profile

All user data is stored in the user profile, which can be viewed by clicking the respective button at the top of the window. In the profile panel, you can change your user information and the app theme, as well as view the results of finished quizzes.

color theme

All colors in aleph are fully customizable. Color values must be full-length hex triplets with preceding #s (such as #F3AA57).

completed quizzes

All completed quizzes appear here. To view the results of a quiz, click on its list entry here.

creating question sets

Question sets are stored as JSON files in the app's assets/questionSets folder. aleph loads all sets found in this folder on launch. Custom question sets can be made following the JSON structure of the example set provided. Excluding true/false questions, all types can have as few or as many answers as may be necessary.
